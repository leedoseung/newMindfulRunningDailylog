디자인 포인트
2025 트렌드 적용 요소
① 크루 스트립
오늘 달린 멤버 → 파란 링 + 이름 + 달린 시간. 아직 안 달린 멤버 → 회색으로 표시. BeReal / Instagram Stories 스타일.
② 소셜 카드 구조
각 카드에 아바타 + 이름 + 시간대를 명시. 기존 masonry에서 세로 단일 컬럼으로 전환 → 인용구가 더 크고 읽기 쉬워짐.
③ 히어로 카드
첫 번째 카드는 다크 배경 + 4.5rem 숫자로 "오늘 달린 시간"을 강조. 나머지는 화이트.
④ 커뮤니티 펄스 카드
이번 주 크루 달리기 현황을 블루 카드로. 요일별 바 차트 + 누적 시간. "혼자 달리는 게 아니다"는 느낌.
⑤ 칩(Chip) 태그
날씨 / 장소 / 순위를 소형 칩으로 표시. 기록에 이미 location 필드가 있어 즉시 적용 가능.
구현 크기 예상
크루 스트립 — 중간 (오늘 달린 사람 쿼리 필요)
카드 리디자인 — 작음 (CSS 교체)
커뮤니티 펄스 카드 — 중간 (주별 집계 쿼리)
칩 태그 — 작음 (기존 location 필드 활용)
